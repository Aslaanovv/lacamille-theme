# La Camille - Luxury Lingerie Shopify Theme

A custom Shopify theme for La Camille - an Egyptian luxury lingerie and pajamas brand.

## Features

- **Parallax Scrolling Intro**: Elegant animated storytelling sections
- **Soft Neutral Color Palette**: Cream, blush, and champagne tones
- **Animated Product Cards**: Details reveal on hover with smooth transitions
- **Fully Responsive**: Optimized for mobile, tablet, and desktop
- **Luxury Typography**: Elegant serif headings with clean sans-serif body text
- **Quick Add to Cart**: One-click add to cart functionality
- **Wishlist Toggle**: Interactive wishlist buttons
- **Color Swatches**: Visual color selection on product cards

## Installation

1. **Compress the theme folder into a ZIP file**
2. **Go to your Shopify admin** → Online Store → Themes
3. **Click "Upload theme"**
4. **Select the ZIP file**
5. **Click "Publish"** on the uploaded theme

## Customization

### Colors

Go to **Online Store** → Themes → Customize → Theme Settings → Colors

- **Cream**: #FDF8F3 - Main background
- **Blush**: #E8D4C4 - Accents and overlays
- **Champagne**: #D4AF37 - Buttons and highlights
- **Text**: #4A4A4A - Main text color

### Typography

- **Headings**: Cormorant Garamond (serif)
- **Body**: Montserrat (sans-serif)

### Homepage Sections

1. **Hero Parallax**: Animated intro with logo reveal
2. **Story Section**: Brand storytelling with parallax
3. **Collections Parallax**: Featured collections
4. **Animated Products**: Product grid with hover effects

## Product Image Guidelines

- **Minimum size**: 800x800px
- **Recommended**: 2048x2048px
- **Format**: PNG or JPG
- **Background**: White or neutral
- **Multiple angles**: Main + alternate images for hover effect
